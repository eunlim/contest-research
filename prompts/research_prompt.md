# 공모전 아이디어 리서치 프롬프트 모음

공모전 유형에 상관없이 재사용할 수 있는 프롬프트입니다.
`[대괄호]` 안을 본인 상황에 맞게 바꿔서 Claude에 붙여넣으세요.

---

## 1. 아이디어 발굴 프롬프트 (전 유형 공용)

```
공모전을 준비 중입니다. 아래 조건에 맞는 아이디어 5개를 제안해 주세요.

- 공모전명: [예: 청년정책 아이디어 공모전]
- 공모전 유형: [공공데이터 / 정책제안 / 창업 / ESG / 마케팅·브랜딩 / 서비스기획 / 기타]
- 주제: [예: 청년 1인가구 고립 문제]
- 제약조건: [예: 실현 가능성 중시 / 예산 최소화 / 6개월 내 출시 가능]
- 원하는 결과물: [예: 정책 보고서 / 앱 프로토타입 / 캠페인 기획안]

각 아이디어에 대해 다음을 포함해 주세요:
1. 제목
2. 핵심 문제 정의
3. 타깃 (수혜자 또는 고객)
4. 해결 방식
5. 필요한 자원 또는 데이터
6. 경쟁 아이디어 대비 차별성
7. 예상 임팩트
```

---

## 2. 아이디어 점수화 프롬프트

```
아래 아이디어를 4가지 기준으로 각 1~10점 채점해 주세요.

- 공모전 유형: [공모전 유형 입력]
- 아이디어 제목: [아이디어 제목]
- 아이디어 설명: [3~5줄 요약]

채점 기준:
- novelty (독창성): 기존 접근 방식 대비 참신한 정도
- feasibility (실현가능성): 주어진 자원·기간 내 실제 구현 가능한 정도
- impact (파급력): 사회적·비즈니스적·정책적 영향력
- differentiation (차별성): 경쟁작 대비 뚜렷이 구별되는 강점

각 항목에 점수(1~10)와 한 줄 근거를 함께 제시해 주세요.
마지막에 총점(4개 평균)을 계산해 주세요.
```

---

## 3. 경쟁작 분석 프롬프트

```
아래 공모전 분야의 기존 수상작 또는 유사 사례를 분석해 주세요.

- 공모전 유형: [공모전 유형]
- 분야: [예: 청년 복지 / 친환경 패키징 / 핀테크 스타트업 등]
- 분석 목적: 우리 팀 아이디어의 차별점을 찾기 위해서

포함할 내용:
1. 대표 사례 3~5개 (수상작 또는 실제 서비스)
2. 각 사례의 핵심 아이디어와 자원
3. 공통적으로 보이는 접근 방식
4. 아직 다루어지지 않은 공백 영역
5. 차별화 제안
```

---

## 4. 보고서 구조 설계 프롬프트

```
공모전 제출용 보고서 목차를 설계해 주세요.

- 공모전 유형: [공모전 유형]
- 주제: [주제]
- 핵심 메시지: [이 아이디어로 전달하고 싶은 것]
- 주요 근거: [데이터, 사례, 인터뷰 등 보유한 근거]
- 분량 제한: [예: A4 10장 이내]

심사위원이 납득할 수 있는 논리 흐름으로 목차와 각 섹션 내용을 구체적으로 작성해 주세요.
공모전 유형별 심사 기준을 반영해 주세요.
```

---

## 5. 최종 점검 프롬프트

```
공모전 제출 전 최종 검토를 해주세요.

- 공모전 유형: [공모전 유형]
- 아이디어 제목: [제목]
- 핵심 내용 3줄 요약: [요약]
- 제안하는 해결 방식: [내용]

아래 관점에서 피드백해 주세요:
1. 논리 흐름의 허점 또는 약점
2. 근거가 부족하거나 과장된 부분
3. 심사위원이 가장 먼저 의문을 가질 지점
4. 경쟁작 대비 차별점이 충분한지
5. 개선을 위한 구체적 제안
```

## 공모전 유형별 핵심 심사 기준 참고표

| 유형 | 가장 중요한 심사 기준 | 주의할 점 |
|------|---------------------|----------|
| 공공데이터 | 데이터 활용 창의성, 시각화 완성도 | 분석 근거 명확히 제시 |
| 정책제안 | 실현 가능성, 정책 파급력 | 법령·예산 근거 필수 |
| 창업 | 시장성, 비즈니스 모델 명확성 | 수익 구조와 고객 정의 |
| ESG | 측정 가능한 환경·사회적 임팩트 | 그린워싱 오해 주의 |
| 마케팅·브랜딩 | 타깃 명확성, 캠페인 실행력 | 예산과 채널 구체화 |
| 서비스기획 | UX 완성도, 사용자 검증 여부 | 프로토타입 또는 화면 설계 포함 |

## 예시 입력 3가지

### 예시 A — 공공데이터형

```
- 공모전명: 공공데이터 활용 창업 경진대회
- 공모전 유형: 공공데이터
- 주제: 고령 운전자 교통사고 예방
- 제약조건: 공개 데이터만 사용, 분석 보고서 형태
- 원하는 결과물: 데이터 분석 보고서 + 정책 제언
```

### 예시 B — 정책제안형

```
- 공모전명: 청년정책 아이디어 공모전
- 공모전 유형: 정책제안
- 주제: 청년 1인가구 고립 문제
- 제약조건: 실현 가능성 중시, 기존 인프라 활용
- 원하는 결과물: 정책 제안서 (A4 10장)
```

### 예시 C — 마케팅·브랜딩형

```
- 공모전명: 대학생 마케팅 전략 공모전
- 공모전 유형: 마케팅·브랜딩
- 주제: MZ세대 친환경 소비 습관 형성 캠페인
- 제약조건: 온라인 채널 중심, 6개월 캠페인 기간
- 원하는 결과물: 캠페인 기획안 + 예산 계획서
```
